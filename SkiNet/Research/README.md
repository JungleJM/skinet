# Research

This directory holds research notes and source-based analyses of external projects that inform SkiNet design decisions.

Use it for:

- comparative reviews of other repos or agent systems;
- implementation notes extracted from primary-source code inspection;
- decision-support documents that are informative, but not part of the active build plan.

Do not use it for:

- current production plans;
- archived SkiNet plan versions;
- run artifacts or controller evidence.
